📖 O programu a ukázky

Mancala je logická desková hra pro dva hráče. Tato digitální verze kombinuje výkonné Python jádro s webovým rozhraním pro přehlednou správu výsledků.

Herní GUI (Pygame)

Hra nabízí plynulé animace, reálnou fyziku semen a možnost nastavení v JSON souboru.

Webová správa (Flask)

Umožňuje sledovat globální statistiky a historii zápasů přes SQL JOIN dotazy.

⚙️ Technická dokumentace

Použité knihovny

Pygame Flask SQLite3 JSON

Klíčový algoritmus (Pseudokód)

FUNKCE proved_tah(index_jamky): pocet_kamenu = jamky[index_jamky] jamky[index_jamky] = 0 -- Vyprázdnění jamky DOKUD pocet_kamenu > 0: posun_se_na_dalsi_jamku() POKUD jamka_je_platná: přidej_kámen(jamka) pocet_kamenu -= 1 POKUD posledni_kamen_v_mancale: hrac_tahne_znovu() -- Bonusový tah POKUD posledni_kamen_do_vlastni_prazdne: zajmi_souperovy_kameny() -- Strategický zisk

Databázové schéma (ER diagram)

Databáze je navržena v Boyce-Coddově normální formě (BCNF) pro eliminaci redundance dat.

Vztah 1:N
Uživatel má více záznamů v účastech v zápasech.
Vztah M:N
Uživatelé a Zápasy jsou propojeny přes vazební tabulku se skóre.
